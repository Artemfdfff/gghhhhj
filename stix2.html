ПОШАГОВОЕ ПРИГОТОВЛЕНИЕ

Продукты для рецепта понадобятся самые простые, у меня всё уже на столе. Филе лучше брать некрупное, чтобы трубочки получились небольшие, аккуратные.
Фото приготовления рецепта: Трубочки из куриного филе с сырным кремом - шаг №2 Каждое филе разрезаем вдоль пополам. Отбиваем до равномерной толщины (под пленкой). Каждую отбивную солим и смазываем растительным маслом. После этого накручиваем отбитое филе на специальную трубочку-конус для выпечки (если трубочки нет, её можно сделать из картона и фольги - свернуть конус из картона, обернуть с двух сторон фольгой).
Фото приготовления рецепта: Трубочки из куриного филе с сырным кремом - шаг №3 Чтобы куриное мясо держалось на трубочке, несколько раз обматываем его ниткой. Ещё раз смазываем курицу растительным маслом.
Фото приготовления рецепта: Трубочки из куриного филе с сырным кремом - шаг №4 Каждую заготовку обваливаем в панировочных сухарях. Выкладываем куриные трубочки в форму для запекания. Помещаем в духовку и запекаем при температуре 200 градусов минут 30.
Фото приготовления рецепта: Трубочки из куриного филе с сырным кремом - шаг №5 Теперь самое время приготовить крем-начинку. Для этого соединяем творожный сыр и сметану. Хорошо перемешиваем. Добавляем мелко нарезанные сладкий перец и укроп. Перемешиваем.
Фото приготовления рецепта: Трубочки из куриного филе с сырным кремом - шаг №6 Куриные трубочки готовы и остывают. Аккуратно убираем нитки (часть сухарей осыплется, ничего страшного).
Фото приготовления рецепта: Трубочки из куриного филе с сырным кремом - шаг №7 Используя кондитерский мешок, наполняем трубочки сырным кремом.
Фото приготовления рецепта: Трубочки из куриного филе с сырным кремом - шаг №8 Получилась оригинальная, интересная и очень вкусная закуска, которая украсит любой праздничный стол. Обязательно попробуйте, если вы любите, чтобы было вкусно и красиво!
